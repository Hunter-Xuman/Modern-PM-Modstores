# Paper Mario 3DS Unistore
An online repository of community made mods for Paper Mario Sticker Star.

# Installation
You will need to install Universal-Updater. https://github.com/Universal-Team/Universal-Updater
Download the source. Take Paper-Mario-3DS.unistore and Paper-Mario-3DS.t3x and place them in sdmc:/3ds/Universal-Updater/stores on your SD card. In Universal Updater, open the settings tab, then go to Select Unistore and select Paper-Mario-3DS-Unistore. From here, you can select the mod(s) you would like to install, and follow the on screen prompts.
